# Sincronización de los READMEs con la v1.17.0

**Fecha:** 2026-07-05
**Área:** docs

## Descripción

Los seis READMEs (es/en/zh-CN/pt-BR/fr/de) no reflejaban la v1.17.0: la
sección "Qué actualizamos" terminaba en v1.16.5 y el ejemplo de instalación
citaba `sftp-1.16.5.vsix`. Se agrega la subsección v1.17.0 (contraseñas
seguras con SecretStorage + CI/release automatizada) en los seis idiomas y
se actualiza el ejemplo de instalación a `sftp-1.17.0.vsix`, cumpliendo la
regla de sincronización de traducciones de `Documentations.md`.

## Tipo de Cambio

- `Cambiado`
