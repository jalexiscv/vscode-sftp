# Release 1.19.0: administrador gráfico de conexiones

**Fecha:** 2026-07-07
**Área:** build

## Descripción

Release menor centrada en el nuevo Administrador de Conexiones (panel webview
para gestionar las conexiones/perfiles de `sftp.json` sin editar el JSON a
mano) y en la activación del modo `strict` de TypeScript. Publicada por el
workflow automatizado al taggear `v1.19.0`.
